Recon

passive & active

Passive

Finding information about the target without interacting with the infra/site

A.K.A OSINT

  • Google dorking
  • Certificate records
  • LinkedIn
  • Shodan

Active

Anything where you are interacting with the infra/site

  • DNS (incl. bruteforcing)
  • Certificates
  • Endpoints
  • Source code
  • Headers

Bruteforcing at Uni
